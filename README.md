Building
--------

You will need the following:

- Wayland
- [swc][0]
  - [wld][1]
  - libdrm
  - libevdev
  - libxkbcommon
  - libinput
  - libudev
  - libxcb (for XWayland)
  - xcb-util-wm (for XWayland)
  - pixman
  - Linux >= 3.12

  [0]: https://github.com/michaelforney/swc
  [1]: https://github.com/michaelforney/wld
